Materialz

This addon adds new materials including ores, metals, alloys, cardboard, flax, and dyes. It expands crafting and exploration features within Cubyz.

⬇ Download v0.2

Features

8 New Metals

  • Tin
  • Aluminum
  • Zinc
  • Lead
  • Steel
  • Rose Gold
  • Brass
  • Bronze

4 New Ores

  • Bauxite (Aluminum)
  • Galena (Lead)
  • Zinc
  • Cassiterite (Tin)

New Materials

  • Wood Pulp
  • Cardboard (+ Cardboard Blocks)

Note: All new metals + cardboard can be used for tool crafting.

New Systems and Items

  • Dyeing system using 20 new dye items
  • New plant: Flax
  • New items: Flax fiber, Ash, Lime, Alkali, Crude Glass Mix
  • Cloth crafting using dyes and flax fiber
  • Nugget crafting (for all ingots in addon + vanilla)
  • Craftable colored glass
  • New block: Crude Glass
